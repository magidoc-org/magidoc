Rollup fetch GraphQL Schema

standalone
plugin
rollup
vite

This rollup plugin fetches the GraphQL Schema required by Magidoc Plugins to build your website documentation. It performs a full GraphQL Introspection Query and stores the result on the file-system. This approach keeps your documentation independent from the actual API.

This plugin is also compatible with ViteJS , which is used by SvelteKit.

Install #

First, add the module to your project.

Then, add the plugin to your Rollup/ViteJS configuration. Here is an example for a svelte.config.js .
